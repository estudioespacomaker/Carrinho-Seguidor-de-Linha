# 🤖 Carrinho Seguidor de Linha

Repositório oficial do projeto **Carrinho Seguidor de Linha** do canal **Espaço Maker**.

Aqui você encontrará todos os códigos, esquemas elétricos, diagramas, modelos 3D (quando disponíveis), materiais de apoio e exemplos utilizados nas aulas do canal ou publicação do blog.

## 📚 Sobre o Projeto

O Carrinho Seguidor de Linha é um dos projetos mais clássicos da robótica educacional. O robô é capaz de identificar uma linha no chão, desviar de objetos e seguir seu percurso automaticamente.

Este projeto foi desenvolvido com foco no aprendizado de:

- Robótica Educacional

É indicado para iniciantes, estudantes, professores e entusiastas do universo Maker.
